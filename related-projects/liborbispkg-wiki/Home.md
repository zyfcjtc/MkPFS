# LibOrbisPkg Wiki

The LibOrbisPkg project consists of a C# Library and a set of tools which utilize the library for creating, manipulating and inspecting PS4 PKG files. Click on the headings for more details.

To download PkgEditor, PkgTool, or LibOrbisPkg, visit the [AppVeyor artifacts page](https://ci.appveyor.com/project/maxton/liborbispkg/build/artifacts).

## [The LibOrbisPkg Library](Library)

You can use this library to add PKG/SFO/RIF/GP4 functionality to your .NET application.

## [PkgEditor](PkgEditor)

PkgEditor is a GUI PKG tool for the PS4. It can open GP4, PKG, and SFO files. It can create GP4, SFO, PFS and PKG files.

## [PkgTool](PkgTool)

PkgTool is the command-line interface PS4 PKG tool. It has options to build and extract PKGs and PFS images, view PKG entries and extract and decrypt files and images from PKGs and PFS.
